# Advanced Line Follower Using PID

## Overview
This project demonstrates motor control and turning behavior using PWM on the MSP432P401R microcontroller. The system forms the foundation for PID-based line following and robotic motion control.

## Features
- PWM-based motor control
- Forward movement and turning behavior
- Adjustable motor speeds
- Basic PID framework implementation
- MSP432P401R embedded system control

## Hardware Components
- MSP432P401R Microcontroller
- TI-RSLK MAX Robot Platform
- DC Motors
- Battery Power Supply

## Software
- Code Composer Studio (CCS)
- C Programming Language
- Timer A0 PWM Control

## System Description
The robot moves forward when both motors operate at the same speed. Turning is achieved by reducing the speed of one motor relative to the other. PWM signals generated by the MSP432 control motor speed and direction.

## Output / Results
The robot successfully demonstrated:
- Forward motion
- Left and right turning behavior
- Controlled motion using PWM speed adjustment

The project verifies the relationship between software-based motor control and physical robotic movement.

## PID Control
A PID control framework was implemented to improve motion smoothness and provide a foundation for advanced line-following behavior.
